Custom widgets such as team member, testimonial, service box and more.
 
== Description ==
 
Primarily meant to extend the functionality of "Meridian One" WordPress theme by MeridianThemes ( pending release ).

**Why do I need this plugin with Meridian One theme:** 

 This plugin adds widgets that are used on the homepage of the theme. The widgets are not part of the theme because widgets are "plugin-territory" and are not allowed in themes.
 
== Installation ==
 
=== From within WordPress ===
1. Visit 'Plugins > Add New'
1. Search for 'Meridian One Features'
1. Activate the plugin from your Plugins page.

=== Manually ===
1. Upload the `meridian-one-features` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
 
== Frequently Asked Questions ==
 
= Why do I need this plugin with Meridian One theme: =

Sure, the custom widgets will work with other themes but the primary purpose is to extend "Meridian One" theme.
